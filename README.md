# 📌 Page Replacement Simulator

## 📖 Project Overview

This project is a Python-based Operating Systems simulator that demonstrates different **Page Replacement Algorithms**. It helps in understanding how memory management works by simulating page faults, hits, and replacement strategies.

The simulator includes a **Graphical User Interface (GUI)** and visualization to compare algorithm performance.

---

## 🎯 Objectives

* To implement and simulate page replacement algorithms
* To compare their performance using page faults and hit ratio
* To provide a user-friendly GUI for better understanding

---

## ⚙️ Algorithms Implemented

* **FIFO (First In First Out)**
* **LRU (Least Recently Used)**
* **Optimal Page Replacement**

---

## 🧩 Modules Breakdown

### 1. Algorithm Module

* Implements FIFO, LRU, and Optimal algorithms
* Calculates page faults and tracks frame history

### 2. Utility Module

* Computes performance metrics like hits and hit ratio

### 3. Visualization Module

* Displays comparison graph of page faults using matplotlib

### 4. GUI Module

* Built using Tkinter
* Takes user input and displays results in table format
